Increment 3 - Menus

  1. Menus can be created:
    1. Menus have a title: Required, text, 120 characters max
    2. A menu belongs to exactly one user
    3. Menus reference at least one recipe of the same user they belong to
    4. The order in which recipes are referenced in the menu is important
    5. If a recipe gets deleted, it is automatically removed from all menus
    6. If a menu contains no more recipes it gets deleted
