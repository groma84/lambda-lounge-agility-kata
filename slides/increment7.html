Increment 7 - Export as HTML

  1. The export of a recipe or menu must be extended with a second format (the first one must still work!):
    1. Export now exports a valid HTML file
    2. Filename is the title of the recipe or menu
    3. If the file already exists it must be overwritten
    4. The title must be a heading
    5. Ingredients are listed as a table
    6. The portion amount the ingredients are for must be shown
    7. The method is a text paragraph
    8. A recipe output must contain how often it is referenced in any menus if a recipe is exported directly
    9. If an image URL is present the image must be embedded in the HTML file
    10. There must be no temporary files left after the export - only the html file
