Increment 5 - Making it social

  1. Users can now like recipes and menus of other users
  2. If a user is deleted all her/his likes are removed
  3. A menu can now also reference recipes from other users
  4. The export of menus and recipes must be extended:
    1. The like count must be shown
    2. If a menu contains a recipe from another user print the name of the user within the recipe
