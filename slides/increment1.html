Increment 1 - Users and recipes

  1. A recipe consists of:
    1. A title: Required, Text, with a maximum length of 120 characters
    2. A cooking method: Required, Text, with a maximum length of 2000 characters
    3. Ingredients: Required, A collection of ingredients, each consisting of:
      1. An amount: Required, Text, includes the unit of measure as text, 60 characters max
      2. An ingredient name: Required, Text, 60 characters max
    4. An image URL: Optional, Text, 255 chars max
  2. A user consist of:
    1. A real name: Required, Text, 60 characters max
    2. A username: Required, only letters a-z, A-Z, 0-9, 16 characters max
  3. Usernames must be unique - trying to create a user with an already existing name results in an error
  4. Recipes belong to exactly one user - trying to attach a new recipe to an invalid user results in an error
  5. Recipes and users can only be created - neither deleted nor edited
  6. Recipes and users are stored in-memory only
