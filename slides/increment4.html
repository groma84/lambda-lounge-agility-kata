Increment 4 - Shopping list

  1. The ingredients format is changed:
    1. The amount is now a floating point number
    2. The unit of measure is now a text with 60 characters max, and:
      1. There is a way to export a list of all already used units of measure in the system (used in any recipe)
    3. The ingredient name is still a required text with 60 characters max
  2. A shopping list can be exported:
    1. A shopping list references menus and/or recipes
    2. A shopping list must reference at least one menu or recipe
    3. Ingredient amounts are rounded to two decimals - trailing zeroes are removed (so "2.0 pc eggs" becomes "2 pc eggs", but "0.33 l milk" stays as "0.33 l milk"
    4. A shopping list adds ingredients with the same name and unit of measure automatically, so "2.0 pc eggs" and "3.0 pc eggs" get exported as "5 pc eggs"
    5. The export contains the added amounts, units of measures and names of all ingredients
  3. The export format is not important - plain text is fine
